# 축구 경기 투표 시스템 PRD

## 1. 개요
- **프로젝트명**: 축구 경기 투표 시스템
- **목적**: 축구 경기 참석 여부를 투표하고 관리하는 웹 애플리케이션
- **주요 기능**: 경기 일정 관리, 참석/불참 투표, 투표 현황 조회

## 2. 기술 스택
### Frontend
- React.js
- Material-UI (MUI)
- date-fns (날짜 처리)
- Axios (API 통신)

### Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- JWT 인증

## 3. 기능 요구사항

### 3.1 사용자 관리
- [x] 회원가입
  - 이메일, 비밀번호, 이름 필수 입력
  - 이메일 중복 체크
  - 비밀번호 암호화 저장

  - 경기장
