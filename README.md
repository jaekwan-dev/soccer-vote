# Football Vote

축구 경기 일정에 대한 투표 시스템입니다.

## 기술 스택

### 백엔드
- Spring Boot 3.2.3
- Spring Security
- Spring Data JPA
- H2 Database
- JWT

### 프론트엔드
- React 18
- Material-UI
- React Router
- Axios

## 시작하기

### 백엔드 실행
```bash
cd soccer-vote
./mvnw spring-boot:run
```

### 프론트엔드 실행
```bash
cd soccer-vote/frontend
npm install
npm start
```

## API 문서
- H2 콘솔: http://localhost:8080/h2-console
- API 엔드포인트: http://localhost:8080/api

## 개발 환경
- Java 17
- Node.js 18+
- npm 9+
